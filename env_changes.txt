КРИТИЧЕСКИЕ ИЗМЕНЕНИЯ В .ENV ФАЙЛЕ:

1. ПОРТ СЕРВЕРА (КРИТИЧНО!):
   Было: PORT=8000
   Нужно: PORT=8055

2. WEBSOCKET ЭНДПОИНТЫ (КРИТИЧНО!):
   Было:
   VAD_WS_ENDPOINT=wss://en.workandtravel.com/vad_realtime/ws
   BUTTON_WS_ENDPOINT=wss://en.workandtravel.com/button_realtime/ws
   BUTTON_UPLOAD_ENDPOINT=https://en.workandtravel.com/button_realtime/upload-audio
   
   Нужно:
   VAD_WS_ENDPOINT=wss://en.workandtravel.com/ws
   BUTTON_WS_ENDPOINT=wss://en.workandtravel.com/ws-button
   BUTTON_UPLOAD_ENDPOINT=https://en.workandtravel.com/api/upload-audio/

3. БЕЗОПАСНОСТЬ КУКИ (для продакшена):
   Было: COOKIE_SECURE=false
   Нужно: COOKIE_SECURE=true

4. DEBUG РЕЖИМ (для продакшена):
   Было: DEBUG=true
   Нужно: DEBUG=false

5. CORS (более строгий для продакшена):
   Было: ALLOWED_ORIGINS=*
   Нужно: ALLOWED_ORIGINS=https://en.workandtravel.com,http://91.218.140.211:8055,http://127.0.0.1:3000,http://localhost:3000

6. WEBHOOK И PAYMENT URL (для продакшена):
   Было:
   WEBHOOK_URL=http://127.0.0.1:8000/api/webhook/payment
   PAYMENT_RETURN_URL=http://127.0.0.1:8000/payment/success
   
   Нужно:
   WEBHOOK_URL=https://en.workandtravel.com/api/webhook/payment
   PAYMENT_RETURN_URL=https://en.workandtravel.com/payment/success

7. DATABASE_PATH:
   Было: DATABASE_PATH=./server/users.db
   Нужно: DATABASE_PATH=./users.db

8. TEMP_AUDIO_DIR:
   Было: TEMP_AUDIO_DIR=./server/temp
   Нужно: TEMP_AUDIO_DIR=./temp

9. ДОБАВИТЬ (если нет):
   NEXT_PUBLIC_API_URL=""  # Пустой, так как API на том же домене
   NEXT_PUBLIC_UPLOAD_URL="/api/upload-audio/"

ПРОВЕРЬТЕ ЭТИ ЗНАЧЕНИЯ - ОНИ УЖЕ ПРАВИЛЬНЫЕ:
   NEXT_PUBLIC_REALTIME_CHAT_CONNECT="/ws"  ✓
   NEXT_PUBLIC_BUTTON_CHAT_CONNECT="/ws-button"  ✓
